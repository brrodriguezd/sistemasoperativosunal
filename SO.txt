-Introduccion de los sistemas operativos 
	Sistema operativo
		Conjunto de programas que controla la ejecución de un software de aplicación, actua como una interfaz entre el usuario y el hardware
		Adminitran los recursos del hardware para el funcionamiento de la coomputadora 
	Objetivo principal
		Facilitar el uso del dispositivo y hacerlo intuitivo 
	Características
		Evolucionar, permite la introducción de nuevas funciones para el sistema
		Adiministrar el hardware
		Relacionar dispositivos
		Organiza datos, para el acceso rápido
		Manejar las comunicaciones de red
		Facilita el uso de los dispositivos de entrada y salida
		Gestión de recuperación de datos
	Tipos de sistemas operativos 
		Multitarea 
			Pueden ejecutar varios problemas a la vez (el más común)
		Monotarea
			Usa todos los recursos en un solo problema
		Monousuario 
			Sólo puede atender a un usuario a la vez
		Multiusuario 
			Puede atender a varios usuarios al mismo tiempo
	Funciones y entorno del trabajo de un sistema operativo

		Entorno
